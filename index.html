Liste mit Booklets, die ich entweder selbst programmiert habe, oder nutze.

Wie kann ich ein JavaScript Booklet hinzufügen?
Füge den Code als URL dem Lesezeichen hinzu.

  1. HTML Tags in Webseite

     javascript:(function(){var script=document.createElement('script');script.src='https://cdn.jsdelivr.net/gh/HolgerKurtz/bookmarklets@latest/bookmarklet.js';document.body.appendChild(script);})();

  2. Platzhalter

INFO:

Die .js Datein werden bei jsdelivery gehostet, da dort – anders als bei rawgit – die neueste Version verwendet wird und nicht die gecachete. raw.github wäre auch möglich, aber Browser blockieren die Ausführung der Datei, da Content Type falsch ist.
